## Running project in your machine

---

1 - Inside project directory install all dependencies (packages) by typing:
`npm install `

2 - Run project on browser by typing:
`npm start `

3 - Go to http://localhost:3000

   <br/>

## For tests

---

1 - In root directory type:
`npm run test `
<br/>
